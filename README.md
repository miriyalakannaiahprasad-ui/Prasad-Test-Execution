# Amazon Selenium Java Tests

## Problem Statement
Automated test cases for Amazon product search and cart functionality.

## Test Cases
1. Navigate to Amazon.com, search for an iPhone, add it to the cart, and print the price.
2. Navigate to Amazon.com, search for a Galaxy device, add it to the cart, and print the price.

## Parallel Execution
Both test cases run in parallel using TestNG suite and HyperExecute YAML configuration.

## Tech Stack
- Language: Java
- Framework: Selenium + TestNG
- Cloud Execution: LambdaTest HyperExecute
